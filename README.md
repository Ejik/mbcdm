# Расширение конфигурации МВС: Детали машин 3.0 для 1С: Управление торговлей 11

Расширение конфигурации **МВС: Детали машин 3.0** для **1С: Управление торговлей 11** представляет собой значительное обновление и развитие предыдущей версии **«МВС: Детали машин 2»**, вобравшее в себя весь её многолетний пользовательский опыт.

Она была создана на базе конфигурации **«1С:Управление торговлей 11»**, что позволяет ей эффективно интегрироваться с современными бизнес-процессами и требованиями пользователей для автоматизации:

- оптово-розничных магазинов
- автосервисов
