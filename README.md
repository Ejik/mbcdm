# МВС: Детали машин 3.0
Расширение конфигурации МВС: Детали машин 3.0 для 1С: Управление торговлей 11 представляет собой значительное обновление и развитие предыдущей версии “МВС: Детали машин 2”, вобравший в себя весь ее многолетний пользовательский опыт. Она была создана на базе конфигурации “1С:Управление торговлей 11”, что позволяет ей эффективно интегрироваться с современными бизнес-процессами и требованиями пользователей для автоматизации оптово-розничных магазинов и автосервисов.
